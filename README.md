Sobre este projeto:

- 🔎 Pesquisar Atores e Filmes
- 🎥 Assistir Trailers
- 🔥 Obter histórico de pesquisa
- 🐱‍👤 Obtenha filmes/programas de TV semelhantes
- 💙 Página de destino incrível
